### 容器级：
```c++

div,h1,li,dt,dd

```




### 文本级：
```c++

p,a,span,em,i,b,u

```

### 可继承的属性：
```c++

text-
line-
font-

```

### 伪类：
```c++

a:link; 	// 未点
a:visited; 	// 点后
a:hover; 	// 悬停
a:active; 	// 点击瞬间

```


### 小标签：
```c++

s,del 	// 删除线
i,em 	// 倾斜
u,ins 	// 下划线

```
